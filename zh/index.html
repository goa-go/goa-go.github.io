Goa

基于中间件的轻量级golang web框架

快速上手 →

轻量

体积小，干净简洁，不绑定任何中间件。

灵活

Goa可以通过中间件方便地扩展功能。

高效

内置context变量池、性能和内存优化，路由基于httprouter二次开发。

# 安装

$ go get -u github.com/goa-go/go

# Hello World

package main

import (
  "log"

  "github.com/goa-go/goa"
)

func main() {
  app := goa.New()
  app.Use(func(c *goa.Context) {
    c.String("Hello World!")
  })
  log.Fatal(app.Listen(":3000"))
}
